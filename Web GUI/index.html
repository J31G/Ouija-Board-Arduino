Welcome!

This is a web application built to control an arduino based ouija board, when connected to the same network as the game board.

In the top right corner, you'll be able to click on the menu button. You'll find the following pages, which I'll give a quick overview below.

  • Home - The page you current are on, giving you an overview of this project.
  • Free Text - This allows you to enter any word that you wish, which then will be send to the ouija board.
  • Predefined - This page is a set of predefined words, which when clicked, will send staight to the board.
  • Config - This allows you to change a few of the varibles in the page, such as speed and wait time.
  • Settings - Within this page, you'll be able to change some of the settings relating to this application, network and connection to the board.
